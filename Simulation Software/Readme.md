# Simulation Software

I don't know how useful this will end up being. This is intended to be a simulated version of the drone that can be used to test the flight controller as it is being developed. May end up being a Python, C#, or Unity application to visualize and control a simulated drone and send/receive signals to/from the flight controller over serial.

Specifically, can be useful for the following and probably more:
- Emulating accelerometer orientations to the flight controller for balance testing.
- Emulating remote control commands to the flight controller.
- Visualizing what the flight controller is trying to do.
